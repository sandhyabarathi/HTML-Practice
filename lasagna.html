World's Best Lasagna

This lasagna recipe takes a little work, but it is so satisfying and filling that it's worth it!

Description

Don’t get me wrong, there are so many ways to make a classic Italian lasagna, but I think people have run too far with the changes, and it no longer resembles anything remotely Italian-style (like so many American-Italian dishes). Even within Italy there are so many variations (more to come)! While lasagna Bolognese is probably the most famous and original, this is a Southern Italian recipe. So if you want a lasagna that’s packed with a laundry list of ingredients including: sausage (with ground beef), ricotta (or worse yet, cottage cheese), sour cream, sugar, Italian seasoning, and other crazy ingredients that you’d never find in authentic Italian lasagna, you better find another recipe, because this is not it.


Ingredients

  • butter
  • flour
  • milk, whole is best
  • dash of nutmeg
  • Kosher or sea salt
  • white pepper

Steps to created:

  1. Make the Meat Sauce
  2. Make the Bechamel Sauce (while the sauce is simmering).
  3. Cook the Noodles (Skip this step if using no-cook lasagna noodles)
  4. Assemble the Lasagna
  5. Bake the Lasagna
  6. SERVE THE DISH!!!
